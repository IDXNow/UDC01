Position Overview

NexusAI Technologies is seeking an experienced Senior Data Engineer to join our Data Platform team. In this role, you'll design, build, and maintain scalable data infrastructure that powers our machine learning systems and analytics platforms.

Responsibilities

  • Design, develop, and maintain scalable data processing pipelines for batch and real-time workloads
  • Architect and implement cloud-based data infrastructure using AWS services
  • Collaborate with data scientists to transform large, complex datasets
  • Build and optimize data warehousing solutions and ETL processes

Required Qualifications

  • Bachelor's degree in Computer Science, Engineering, or related field
  • 5+ years of experience in data engineering or similar role
  • Expert knowledge of Python and SQL
  • Strong experience with data processing frameworks (Spark, Flink, or similar)
  • Experience with AWS cloud services (S3, Redshift, Glue, EMR, Lambda)

Technical Skills

Python SQL AWS Spark Airflow ETL Data Warehousing Kafka Docker Git Terraform CI/CD
